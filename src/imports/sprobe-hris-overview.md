# Sprobe HRIS

Sprobe HRIS is a Human Resource Information System designed to manage employee records, probation tracking, time-off monitoring, and HR analytics through a centralized web dashboard.

The platform provides HR teams with tools to manage employee data, monitor employee status, track probation progress, and oversee workforce metrics.

---

## Overview

Sprobe HRIS provides:

- Employee record management
- HR dashboard analytics
- Probation monitoring
- Leave / time-off tracking
- Employee lifecycle management
- Administrative HR tools

The system is accessible through desktop and mobile browsers.

---

## System Screens

### Login

Secure authentication for administrators and HR personnel.

Features:

- Email login
- Password authentication
- Password recovery
- Session management

---

### Dashboard

The dashboard provides a quick overview of company HR metrics.

Widgets include:

- Total number of employees
- Time-off summary
- Calendar overview
- Probationary progress tracker

Dashboard Components:

| Component | Description |
|-----------|-------------|
| Employee Count | Displays the total number of employees |
| Time Off | Shows employees currently on leave |
| Calendar | Displays upcoming events, meetings, and holidays |
| Probation Progress | Tracks probation stages of employees |

---

### Employee Management

Manage employee records from a centralized table.

Features:

- Add new employee
- Edit employee information
- Delete employee
- Search employees
- Import employee records
- Export employee data

Employee Fields:

| Field | Description |
|------|-------------|
| Employee ID | Unique identifier |
| First Name | Employee first name |
| Last Name | Employee last name |
| Status | Regular / Probationary |
| Date Hired | Employee hire date |

Actions:

- View employee
- Edit employee
- Delete employee

---

### Probation Tracking

Track employees during their probation period.

Probation stages include:

- Just Started
- 2nd Month
- 3rd Month Evaluation
- 4th Month
- 5th Month
- End of Probation Evaluation

Each employee record includes:

| Field | Description |
|------|-------------|
| Full Name | Employee name |
| Email | Employee email |
| Status | Current probation stage |

This helps HR teams ensure probation reviews happen on time.

---

### Calendar

Displays important HR events including:

- Employee evaluations
- Meetings
- Holidays
- Internal HR activities

Calendar Legend:

- Blue: Today
- Red: Meeting
- Other indicators may represent internal events.

---

### Time Off Monitoring

Displays employees currently on leave.

Capabilities:

- Track approved leaves
- View time-off summaries
- Monitor employee availability
